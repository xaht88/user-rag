# RAG в веб приложении. 

## Данное приложение расширяет возможности использование LLM.
	На данный момент LLM при недостатке контекста или при больших его объемах 
галлюцинирует. Подход RAG нивилирует данный эффект, по подготовленной БД моделью типа BERT из загруженного осуществляется поиск запросов пользователя и посредствой выбранной 
LLM предостовляется ответ. 


## Целевая аудитория пользователи компьютера и мобильного телефона.
	Данное приложение сократит время на изучение документов

## Ключевые функции
	Выбор доступных LLM.
	Загрузка файла для анализа с переработкой в эмбединги. 
	Пользовательский интерфейс для работы с загруженными документами.
